handshake_params::handshake_params

Initializing constructor.

Synopsis 
handshake_params(
    boost::string_view username,
    boost::string_view password,
    boost::string_view db = "",
    collation connection_col = collation::utf8mb4_general_ci,
    ssl_mode mode = ssl_mode::require,
    const buffer_params& buffer_config = {});
Description
Parameters

Name

Description

username

User name to authenticate as.

password

Password for that username, possibly empty.

db

Database name to use, or empty string for no database (this is the default).

connection_col

The collation to use for the connection. Impacts how text queries and prepared statements are interpreted. Defaults to collation::utf8mb4_general_ci.

mode

The ssl_mode to use with this connection; ignored if the connection does not support SSL.

buffer_config

Configuration about internal buffers.
