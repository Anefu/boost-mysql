connection::prepare_statement (2 of 2 overloads)

Prepares a statement server-side.

Synopsis 
void
prepare_statement(
    boost::string_view stmt,
    statement< Stream >& result);
Description

After this operation completes, result will reference *this. It will be usable for server interaction as long as I/O object references to *this are valid.

This operation involves both reads and writes on the underlying stream.
